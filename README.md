# Basílica San Vicente Ferrer – Web App (PWA)

## Ejecutar local
- Python:
  python -m http.server 5500
  Abrir http://localhost:5500

## Subir (Netlify)
- Drag & drop la carpeta del proyecto (o el zip descomprimido) en Netlify.

## Editar contenido
- Canciones: data/songs.json
- Oraciones: data/prayers.json
- Santos: data/saints.json
- Historia: data/history.json
- Horarios: data/schedule.json
- Novedades: data/announcements.json

## Imágenes
- assets/img/logo.png (logo)
- assets/img/*.png (flyers)
